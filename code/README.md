# The Daily Grind | Coffee News

A modern, responsive web application that aggregates the latest news from the world of coffee using the NewsAPI.

## Features
- **Live News Fetching:** Real-time updates on coffee trends.
- **Search Functionality:** Search for specific terms like "Espresso," "Latte," or "Roasting."
- **Responsive Design:** Optimized for mobile, tablet, and desktop viewing.
- **Secure Backend:** Uses Vercel Serverless Functions to protect API keys.

## Tech Stack
- **Frontend:** Vanilla HTML5, CSS3 (Grid/Flexbox), JavaScript (ES6+).
- **Backend:** Node.js (for local testing) and Vercel Functions.

## Local Setup
1. Clone the repository.
2. Create a `.env` file and add your `NEWS_API_KEY`.
3. Run `npm install`.
4. Run `node server.js` to start the local proxy.
